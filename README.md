# pai-gent

A resizable multi-panel layout built with Next.js and react-resizable-panels.

## Layout

```
 [A |  B  |  C  |  D  | E ]
 48px    ~33%   ~33%   ~33%   48px
```

- **A / E** — fixed-width toolbars (48px)
- **B / C / D** — flexible panels, draggable separators between them

## Development

```bash
bun dev       # http://localhost:3000
bun build     # production build
bun start     # production server
```
